Introduction

PHP (Hypertext Preprocessor) is a widely-used, open-source, server-side scripting language. It is especially suited for web development and can be embedded into HTML. PHP scripts are executed on the server, and the result is sent to the browser as plain HTML.

You can use PHP to create dynamic page content, collect form data, send/receive cookies, and interact with databases. PHP is known for its simplicity and flexibility.

What You Should Already Know

Before learning PHP, it helps to have:

  • A basic understanding of the Internet and the World Wide Web (WWW)
  • A solid knowledge of HTML
  • Some programming experience (optional but helpful)
PHP and HTML

PHP can be embedded directly into HTML code. The PHP code is enclosed in <?php ... ?> tags and executed on the server.

Example:

<!DOCTYPE html>
<html>
<body>

<?php
  echo "Hello, World!";
?>

</body>
</html>

When this script runs, the browser will receive only the text:
Hello, World!

  $age = 25;
  $isActive = true;

Variable Scope
  • Global: Declared outside functions, accessible everywhere.
  • Local: Declared inside a function, accessible only within that function.

To access a global variable inside a function, use the global keyword:

<?php
  $greeting = "Hello";

  function sayHello() {
    global $greeting;
    echo $greeting;
  }
?>
Constants

You can define a constant using define() or const. Constants cannot be changed once set.

<?php
  define("PI", 3.14);
  const TAX = 0.2;
?>

Constants are automatically global.

Data Types

PHP supports the following data types:

  • String - "Hello"
  • Integer - 42
  • Float (Double) - 3.14159
  • Boolean - true or false
  • Array - array(1, 2, 3)
  • Object
  • NULL
if...else Statement

PHP uses standard control structures like if, else, and elseif.

<?php
  $age = 20;

  if ($age >= 18) {
    echo "Adult";
  } else {
    echo "Minor";
  }
?>
while Loop
<?php
  $i = 0;

  while ($i < 3) {
    echo $i;
    $i++;
  }
?>
Functions

Functions are declared using the function keyword.

<?php
  function square($number) {
    return $number * $number;
  }

  echo square(5); // Outputs 25
?>

Note: Primitive types (like numbers) are passed by value. Objects and arrays can be passed by reference if needed.

PHP Superglobals

PHP provides several built-in superglobal variables which are accessible from anywhere in the script:

  • $_GET - Data from the URL
  • $_POST - Data from a form
  • $_SERVER - Server and execution information
  • $_SESSION - Persistent session data
  • $_COOKIE - Data stored on the client
Handling Forms in PHP

PHP can retrieve form data using the $_GET or $_POST superglobals.

HTML:

<form method="post" action="welcome.php">
  Name: <input type="text" name="name">
  <input type="submit">
</form>

PHP (welcome.php):

<?php
  $name = $_POST['name'];
  echo "Hello, $name";
?>
Connecting to a Database

PHP can connect to MySQL using mysqli or PDO.

<?php
  $conn = new mysqli("localhost", "username", "password", "myDB");

  if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
  }
  echo "Connected successfully";
?>
